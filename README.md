# HR Manager

A task to build a dashboard to manage HR user details. Built with `React` & `Nest`.

## Installation

1. **Install dependencies**:

To install the dependencies on both `front-end/` & `back-end/`

```sh
yarn install-deps
```

2. **Run the project**:

```sh
yarn start
```

2. **Build the project**:

```sh
yarn build
```
